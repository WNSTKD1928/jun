함수와 이름조각

가장 근본적인 린프로그렘의 조각은 이름조각 속으로 함수를 조직화 하는 것
함수는 투입값을 결과 값으로 생성하며, 이름 조각의 기반아래 조직화하는 것이다. 
이는 대상을 그룹화 하는 린의 근본적인 방법이다.
<def>이란 명령어를 이용해 정의한다. 이는 함수의 이름을 주며, 그것의 논의를 정의한다.

'eval'명령어는 표현을 평가하고, 결과를 내놓는다.

함수 정의하기 : def function x := f(x) 
fuction 앞뒤로 임의의 식별이름 붙이기 가능, 변수에 하나의 자연수만을 수용한다. 그리고 결과값도 자연수로 내놓는다.

함수 적용하기 : def result := function 숫자
함수에 투입값을 설정한다. ":="이후 적혀 있는 함수의 이름이 같아야 인식이 된다.

결과값 계산
#eval println! "{result}"
따옴표 안의 내용을 그대로 반영하기(print) 때문에 따옴표 안에는 영/한 언어 둘다 적어도 된다.
다만 대괄호 안의 이름이 이전에 결과를 정의한 이름과 일치해야 한다.

필요한 경우 변수에 이름을 단 채로 함수를 정의할 수 있다.
def function (x : Nat) := f(x)

조건문 설정
def sampleFunction3 (x : Int) :=
  if x > 100 then
    f(x)
  else
    g(x)

이후엔 result 정의 후 결과 값 확인

* #eval println "{function 숫자}" << 형식으로 바로 결과를 도출하여 확인도 가능하다.
